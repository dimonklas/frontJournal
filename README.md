CASHCHARGE:	
dskweb ra

IDENTPHYS:	
riscmg ii, hd

KOPILKA:
dskweb n1, n2, n3; riscmg uk

UNIPACKSAS:
dskweb x1; riscmg nu, iu, hd

UNIREQ:
dskweb ig; riscmg ai
